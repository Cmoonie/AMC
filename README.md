# Spider - AMC

Een AI-gestuurde zoektool voor onderzoeksexpertise binnen Division 9 van Amsterdam UMC.

## Wat doet deze app?
- Zoek op naam, expertise of project
- Vind onderzoekers en hun verbindingen
- AI genereert automatisch een samenvatting van de resultaten

## Technologie
- Python
- Streamlit
- Pandas
- Groq AI (llama-3.3-70b)

## Hoe starten?
1. Installeer de dependencies: `pip install streamlit pandas groq python-dotenv`
2. Maak een `.env` bestand aan met je `GROQ_API_KEY`
3. Start de app: `py -m streamlit run app.py`
